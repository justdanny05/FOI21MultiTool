|              ADMIN OBERFLÄCHE (Benutzer)            |
|                                                     |
=======================================================

Was möchten Sie tun?
1 - Warenkorb anzeigen - Zeigt den aktuellen Warenkorb an.
2 - Produkt hinzufügen - Fügt ein neues Produkt zur Liste hinzu.
3 - Produkt Preis ändern - Ändert den Preis eines vorhandenen Produkts.
4 - Produkt löschen - Löscht ein vorhandenes Produkt aus der Liste.
5 - Produkte Liste - Zeigt eine Liste aller verfügbaren Produkte an.
6 - Automatisch Angemeldet bleiben - Aktiviert die automatische Anmeldung für Benutzer.
7 - Rechte vergeben - Vergibt Berechtigungen an Benutzer.
8 - Benutzer Abmelden - Meldet den Benutzer vom System ab.
9 - Programm beenden - Beendet das Kassensystem.
